# モデル

Video Background Remover CLI は `rembg` が提供する複数の AI モデルに対応しています。`--model` オプションで切り替えることができます。

## 使用可能なモデル

| モデル | 説明 |
| --- | --- |
| `isnet-general-use` | 汎用モデル。ほとんどのユースケースで推奨されるデフォルト設定。 |
| `u2net` | サリエントオブジェクトの抽出に適しています。 |
| `u2netp` | `u2net` の軽量版。高速ですが精度は若干低下します。 |
| `u2net_human_seg` | 人物 / 人体のセグメンテーションに最適化されています。 |
| `silueta` | 高品質な出力が得られますが、処理速度は遅めです。 |

## 使い方

`--model` フラグでモデル名を指定します。

```bash
python main.py input.mp4 output.mp4 --model u2net_human_seg
```

## 注意事項

- モデルは**初回使用時**に自動でダウンロードされ、ローカルにキャッシュされます。
- ダウンロードには通信環境によって時間がかかる場合があります。
- デフォルトの `isnet-general-use` はほとんどの動画に対して良好に動作します。
- 人物が映る動画には `u2net_human_seg` を使用するとより良い結果が得られます。
